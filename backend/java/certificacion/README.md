# SERVICIO PARAMETROS - FENIX
Servicio API REST, que permite gestionar:
- Partidas presupuestarias
- Certificaciones
- Procesos de contratacion (solo recupera una lista de procesos de contratacion) 

## Tecnologías
* Spring Boot 2.4.0
* Java 8
* Maven 3.8.2
* PostgreSQL 10.16 
* OpenAPI 3.0.3
