# Fenix (Front-End)

Este proyectos fue generado con [Angular CLI](https://github.com/angular/angular-cli) version 11.2.14.

## Instalación

Ejecutar `npm install` para crear el scaffolding del proyecto (Ver sección `Code scaffolding`) 

## Ambiente de Desarrollo 

Ejecutar `ng serve` en algun ambiente de desarrollo, luego puede navegar en `http://localhost:4200/`. La aplicación aparecerá automaticamente.

## Code scaffolding

Ejecute `ng generate component component-name` para generar un nuevo componente. Puede usar también `ng generate directive|pipe|service|class|guard|interface|enum|module`.

## Building: Construcción de "ejecutables"

Ejecutar `ng build` para construir una versión ejecutable del proyecto. Esta versión ejecutable estará contenida en la carpeta `/dist`. Use la opción `--prod` para una versión de producción.
